# ✏️ Whiteboard

A beautiful, minimal whiteboard web app — single HTML file, zero dependencies.

## Features

- **8 Tools**: Pen, Highlighter, Line, Arrow, Rectangle, Circle, Text, Eraser
- **6 Colors**: Dark, Red, Blue, Green, Orange, Purple
- **4 Stroke Sizes**
- **Undo / Redo** (up to 60 steps)
- **Save as PNG** (with white background)
- **Keyboard shortcuts**
- **Touch support** (mobile/tablet)
- **Custom cursor ring**
- Beautiful warm paper aesthetic

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `P` | Pen |
| `H` | Highlighter |
| `L` | Line |
| `A` | Arrow |
| `R` | Rectangle |
| `C` | Circle |
| `T` | Text |
| `E` | Eraser |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |

## Usage

Just open `index.html` in any modern browser. No build step, no dependencies.

```
open index.html
```

Or serve locally:

```bash
npx serve .
# or
python3 -m http.server 8080
```
